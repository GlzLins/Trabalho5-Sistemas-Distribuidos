## 🎯 Sobre o Projeto
Este repositório contém a implementação da **Opção D: Algoritmos de Eleição Distribuída**, utilizando o **Algoritmo do Valentão (Bully Algorithm)**. 

A estratégia foi desenvolvida para simular a eleição autônoma de um novo nó coordenador em um sistema distribuído de catracas de reconhecimento facial do Restaurante Universitário (RU), garantindo a continuidade do serviço caso o servidor principal sofra uma falha.

## 📂 Arquivos do Repositório
* 🐍 `bully_election.py`: Código-fonte da implementação usando Sockets UDP e Threads em Python.
